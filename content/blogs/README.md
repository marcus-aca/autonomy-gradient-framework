# Blog Submissions

This directory accepts blog submissions via pull request.

## Required File Format

- Markdown (`.md`)
- Frontmatter block
- Clear section headings (`## Heading`)

## Frontmatter Fields

- `title` (required)
- `excerpt` (required)
- `summary` (required)
- `modeTag` (optional: Assisted | Integrated | Validated | Autonomous | Self-Optimizing | Cross-Gradient)
- `readingTimeMinutes` (optional, integer)
- `author` (optional)
- `keywords` (optional comma-separated string)

Use `_template.md` to start.

## Review Notes

- Align wording to canonical model language in `../../model/terms.yml`
- Prioritize practical journeys and concrete outcomes
- Avoid sensitive/private information
